【昭通市工商局机关党委组织开展“宣传十九大 学习新党章”主题党日活动】

为认真学习宣传贯彻党的十九大精神，结合11月1日的党费日活动，昭通市工商局机关党委组织各支部开展以“宣传十九大、学习新党章”为主题的“主题党日”活动。

11月1日上午九点，市局机关四个在职党支部在家党员分别开展主题党日活动。机关各支部党员顺序排队，自觉缴纳11月份的党费，各支部组织委员在党费证上认真填写、记录。在学习新党章环节，各支部结合实际，多措并举，学习异彩纷呈。机关一支部书记王和梅自己制作PPT课件，带领支部党员共同学习，市局党组书记，局长王剑竹同志以普通党员身份参加一支部主题党日活动，并就学习新党章体会和与会党员共同交流。在机关二支部，支部书记吴琦同志带领与会党员重点学习了“什么是习近平新时代中国特色社会主义思想”和党章修改的十三个重点，学习完毕，组织支部党员交流，老党员周成林同志在发言中说到“作为一名老党员，就是要不忘初心，对党忠诚；落实在行动上，就是要切实履行好自己的工作职责”；在机关三支部，支部书记温纯刚带领与会党员对比学习新旧党章异同点，大家一边学习一边讨论，颇有收获；市局党组成员、副局长肖才志，市局党组成员、副局长柯猛两位同志以普通党员身份参加三支部主题党日活动，并不时和与会党员交流学习体会；在机关四支部，支部书记刘邦彬带领与会党员共同学习新党章的重点内容，针对修改内容，要求支部党员不满足集体学习，会后加强自学，遵照落实。

本次主题党日活动，是昭通市工商局推进“两学一做”学习教育常态化制度化的一个有力抓手，对于进一步增强支部生活的生机活力具有十分重要的意义。在本次主题党日学习活动前，市工商局党组已经利用党组中心组学习时，专题学习了党的十九大会议精神。市局离退休支部在重阳节组织慰问活动时，离退休支部书记李建同志带领市工商局离退休老党员专题学习了党的十九大精神，补足精神之钙，永远跟党走，通过上述措施，确保市工商局所有党员同志学习十九大精神和新党章全覆盖，一个不掉队；下步市工商局还将进一步采取措施，结合工商工作职能，切实落实十九大会议精神。

 

（供稿： 昭通市工商局机关党委  马 娅）
